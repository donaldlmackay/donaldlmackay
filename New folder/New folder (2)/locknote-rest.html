Locknote from Steganos

Locknote from Steganos is an easy to use way of storing passwords etc securely.

It "...enables you to store text information, using ... encryption. You can type any text you want, e.g your bank account information, website passwords, social security number etc. and then simply close LockNote, at which time you will be prompted to set a password. Subsequent access to LockNote requires the correct password...You can copy the locked file to a USB stick or email and unlock it on any PC...".

It is widely available and you can download it from snapfiles.
