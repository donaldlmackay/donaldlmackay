Multi View Notepad++

Notepad++ multi-view enables you to view 2 documents at the same time (possibly the same document). They appear in tabs side-by-side.

Right clicking on the divider allows layout to orient horizontally or vertically.

You can then use the Notepad++ compare plugin

Compare 2 files in Notepad++ with compare plugin

Notepad++ compare plugin allows comparison of the files in side-by-side tabs. You can easily spot any highlighted differences.
