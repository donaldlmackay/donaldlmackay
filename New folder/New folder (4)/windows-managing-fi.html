Windows (7 & Vista) Managing Files

Explorer substitutes

  • ExploreXP - (freeware). Has the great advantage of folder and file size and is interoperable with Windows Explorer. It can copy and paste, and drag and drop both to and from Windows Explorer.
  • There are others such as CubicExplorer and xplorer2 lite - but the benefits are not obvious.
  • There are some circumstances where Windows Explorer (vista) is very slow compared to Windows Explorer (XP) - particularly bulk move and bulk delete from multiple sub folders. A useful tool to overcome this is "Rename Us" which has move and delete builtin to it.

Searching in Windows 7

Renamers

  • Renamer6.0 - (freeware). Fast to load, supports macros.
  • RenamerNG - (freeware). Development of Renamer6.0, supports macros with regular expressions, but not as fast as Renamer6.0.
  • Flexible Renamer - (freeware). Can change case, add folder name to filename etc.
  • Rename Us - (freeware). Actually used for batch delete.

Backup & Download

  • Computer backup: automated local or online
  • SyncBack - (freeware). Backup to e.g. external hard drive.
  • JDownloader - (freeware). Download management tool.

File and string manipulation utilities

  • ls - (freeware). Produces a text file which list the files in folder and a lot more.
  • hjsplit - (freeware). Breaks a file up into pieces. Either by number of lines or bytes.
  • Tabs To Spaces (CLR) - (shareware). Converts Tabs To Spaces. Useful for pasting an extracted spreadsheet into a text file and maintaining columnar layout
  • WinXs - (shareware). Unix style file and string manipulation utilities for Windows.

Misc

  • PowerArchiver-2002 - (shareware). Compresses to zip or unpacks from zip, lots of files in one shot
  • TreeComp - (freeware). Compare 2 directory trees, show differences and optionally synchronize the files within the directories.
